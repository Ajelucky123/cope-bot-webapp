# COPE Referral Bot - Web App

This is the Web App for easy wallet connection in Telegram.

## GitHub Pages Deployment

This webapp is designed to be hosted on GitHub Pages.

### Quick Setup

1. Create a new GitHub repository
2. Push this `webapp` folder to the repository
3. Enable GitHub Pages in repository settings
4. Your URL will be: `https://YOUR_USERNAME.github.io/REPO_NAME/index.html`

### Files

- `index.html` - Main web app file (must be in root or webapp folder)

### Customization

You can customize the web app by editing `index.html`:
- Change colors and styling
- Add more wallet options
- Modify the UI/UX

### Security

- All signatures are verified on-chain
- Telegram user ID is verified
- No private keys are transmitted
